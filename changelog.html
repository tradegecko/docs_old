Apr 03 2014
  • Variant endpoint is now returning 'price_lists' array with information previously returned inside prices object.
Apr 03 2014
  • Variant endpoint is now returning 'locations' array with information previously returned inside stock_levels & committed_stock_levels attributes.
Apr 03 2014
  • Variant endpoint 'stock_levels' is deprecated with intent to remove on April 21
Apr 03 2014
  • Variant endpoint 'committed_stock_levels' is deprecated with intent to remove on April 21
Apr 03 2014
  • Order endpoint 'invoice_numbers' is deprecated with intent to remove on April 21
Apr 03 2014
  • Order endpoint is now returning 'invoices' as a an array, reverts behaviour change from March 03.
Apr 01 2014
  • API Collection endpoints now ignore query parameters with blank values.
Mar 12 2014
  • Product now has `brand` attribute.
Mar 03 2014
  • Order endpoint is now returning 'invoice_numbers' as a Hash of id => invoice_number instead of array.
Feb 17 2014
  • Deprecation: Order `cached_total` attribute is now `total`
  • Deprecation: PurchaseOrder `cached_total` attribute is now `total`
Jan 23 2014
  • Order `source_id` attribute returns the internal id of the OAuth Application
  • Deprecation: Order `url` attribute is now `source_url`
  • Deprecation: Order manually specified `source` attribute has been removed and is replaced by the read-only `source_id`
Jan 22 2014
  • Deprecation: Contact `phone` attribute is now `phone_number`
Jan 16 2014
  • Order now has `ship_at` attribute.
  • Deprecation: Order `due_at` attribute is deprecated and for backwards compatibility, in the interim `due_at` is an alias for `ship_at`.
  • Consumers can now register multiple redirect URIs. Specify the one you want to use as a parameter in your call to https://api.tradegecko.com/oauth/authorize.
Jan 14 2014
  • TradeGecko Applications now have `settings_url` and `icon` attributes. Visit http://go.tradegecko.com/oauth/applications to update your application details.
Jan 03 2014
  • Order endpoint is now returning 'invoice_numbers' as array instead of string.
Dec 20 2013
  • Payment Term now has 'status' attribute and can now be deleted.
